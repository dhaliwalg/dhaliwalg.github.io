Projects

Featured projects that I've worked on!

iOS Development has always been something that I've wanted to do. I've been jailbreaking my iPhone/iPod ever since RedSn0w, which was I think iOS 6 or 7. I just wanted to have cool animations when I was swiping across the screen, which led to me installing the tweak "Barrel". Since then, I've been jailbreaking whenever possible and always wondering how these tweaks were made. I wanted my tech to stand out, as tech is a part of who we are. And we're not as bland as Apple's stock iOS. After buying an old iPhone 6S, I'm able to develop these tweaks in Obj-C. The tweaks are barebones for now, but as I gain experience in Obj-C and developing, I hope to release more complex and diverse tweaks.

This isn't a comprehensive list of my projects Those mentioned on my resume won't be here, and neither will anything that we coded for class. These can be requested by contacting me, and I'll send a GitHub repo. These will mostly be my iOS runtime modifications ("tweaks") and whatever else I develop. It's a pinch difficult to self-learn tweak development, but I won't let that stop me!

    Tweaks I've Made:

  • NoDock

    • This tweak does exactly what the title implies. It takes the dock on the iOS springboard, and then removes it. This is the first tweak I've made, it's very simple. It hooks into the dock and changes the opacity. It goes from looking like this to looking like this after the dock is hidden. This is on my repo, and the source code can be found here.

  • NoLabels

    • UPCOMING! This tweak is also going to be more on the simple side, simply hiding all of the app labels. I'll probably have to hook into the app labels and change the opacity as well. This is going to be my second tweak, just tinkering with the springboard. I will update this page with screenshots when I've built the tweak.
